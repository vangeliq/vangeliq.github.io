# Instructions

#### Prereqs and installation
Prereqs:
- [Node.js](https://nodejs.org/)

Clone and Install repo:
```bash
git clone https://github.com/vangeliq/vangeliq.github.io.git
cd vangeliq.github.io
npm install
```

#### Running with npm

To start the development server, run `npm run dev`
